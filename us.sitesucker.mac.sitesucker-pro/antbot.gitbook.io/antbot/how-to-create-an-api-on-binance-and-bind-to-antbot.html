How to create an API on Binance and bind to AntBot?

1. After opening the Binance APP, click [More] on the [Home] interface.
​
2. Scroll down to the bottom of the [Services] page and find [API Management].
​
3. After entering the API management page, set an API key (the API name, it is recommended to name it AntBot), and click [Create API].
​
4. After entering the security verification page, click [Get code], enter the email verification code, mobile phone verification code and Binance/Google Authenticator code, and then click [Submit].
​
5. After the creation is successful, you must take good care of your Secret Key. The Secret Key will only appear once. At the same time, for the security of your account, please do not share this page. You can copy the Secret Key to AntBot (Home, API KEY) and paste it, and then copy the API key.
​
Note that only 3 permissions of the API are required: Enable Reading, Enable Spot & Margin Trading, and Enable Futures. Do not open other permissions. (If you forget the Secret Key during later use, in order to ensure account security, you can only recreate it by deleting the API)
