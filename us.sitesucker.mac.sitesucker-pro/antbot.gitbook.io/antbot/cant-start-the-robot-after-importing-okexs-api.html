Can’t start the robot after importing OKEx’s API?

After AntBot imported OKEx API, the robot could not be started. First of all, you need to check whether there are funds in the OKEx trading account, and you need to select the trading permissions when creating the API.
Next, you need to go to OKEx to check whether the account mode is in "Single currency margin". By default, new registered users are "Simple" and you need to modify it to "Single currency margin".
Note: When you switch account mode, it prompts "Your settings failed as you have positions, open orders or trading bots.", it means that you currently have a futures position, and you need to close it before you can change it.
