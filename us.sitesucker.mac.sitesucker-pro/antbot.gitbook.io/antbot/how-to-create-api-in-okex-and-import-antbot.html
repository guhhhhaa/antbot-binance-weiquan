How to create API in OKEx and import AntBot?

Before starting, it is recommended that you switch to the English version of OKEx. This is convenient and accurate to check API information.
The first step is to create an API and follow the steps below.
Open OKEx and click on the small avatar in the upper left corner.
Click the API menu to enter the configuration page.
Click Create V5 API key.
Enter the information according to the instructions, remember to select the transaction authority.
Finally, according to the prompts on the API creation page, after entering the information correctly, you can submit it successfully.
